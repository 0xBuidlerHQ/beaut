# useBigIntInput

Keeps a controlled input value synchronized with a `bigint` representation while enforcing decimal precision rules, input validation, and an explicit error flag so conversions never throw.

## Usage

```tsx twoslash
import React from 'react';
// ---cut---
import { useBigIntInput } from "@0xhq/beaut/react";

const { value, valueAsBigInt, handleInputChange, isError } = useBigIntInput(); // [!code hl]
<input value={value} onChange={handleInputChange} />
```
