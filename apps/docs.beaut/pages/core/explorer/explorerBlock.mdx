# explorerBlock

Builds a block URL for a given explorer by normalizing the base URL, validating that the block number is a positive integer, and appending the configured path segment.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const blockUrl = Beaut.explorerBlock("https://etherscan.io", 12345678); // [!code hl]

Output: "https://etherscan.io/block/12345678"
```
