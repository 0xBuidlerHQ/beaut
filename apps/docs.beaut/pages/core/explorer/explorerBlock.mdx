# explorerBlock

Formats a safe `explorer URL` that points to a `block`.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.explorerBlock("https://etherscan.io", 12345678); // [!code hl]

Output: "https://etherscan.io/block/12345678"
```
