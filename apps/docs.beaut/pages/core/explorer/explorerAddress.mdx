# explorerAddress

Formats a safe `explorer URL` that points to an `address`.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.explorerAddress("https://etherscan.io", "0x1234567890abcdef1234567890abcdef12345678"); // [!code hl]

Output: "https://etherscan.io/address/0x1234567890abcdef1234567890abcdef12345678"
```
