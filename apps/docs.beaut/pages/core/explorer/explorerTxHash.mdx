# explorerTxHash

Formats a safe `explorer URL` that points to a `transaction hash`.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.explorerTxHash(
	"https://etherscan.io",
	"0xabcdef1234567890abcdef1234567890abcdef1234567890abcdef1234567890",
); // [!code hl]

Output: "https://etherscan.io/tx/0xabcdef1234567890abcdef1234567890abcdef1234567890abcdef1234567890"
```
