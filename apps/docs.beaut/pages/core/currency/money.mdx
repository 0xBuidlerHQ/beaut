# money

Formats numeric values into localized currency strings using `Intl.NumberFormat`, with defaults for USD/en-US plus overrides for locale, currency, number-format details, and a safe fallback when inputs are invalid.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formattedPrice = Beaut.money(1234.56, { currency: "EUR" }); // [!code hl]

Output: "€1,234.56"
```
