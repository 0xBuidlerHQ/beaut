# money

Formats `numbers` into localized currency strings.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.money(1234.56, { currency: "EUR" }); // [!code hl]

Output: "€1,234.56"
```
