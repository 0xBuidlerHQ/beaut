# hash

Condenses a 66-character transaction hash into a human-friendly string by keeping a configurable prefix and suffix while returning an empty string for malformed data.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const shortHash = Beaut.hash("0xabcdef1234567890abcdef1234567890abcdef1234567890abcdef1234567890"); // [!code hl]

Output: "0xabcd...7890"
```
