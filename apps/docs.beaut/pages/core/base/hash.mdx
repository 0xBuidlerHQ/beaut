# hash

Formats a `hash` into a human-friendly string.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.hash("0xabcdef1234567890abcdef1234567890abcdef1234567890abcdef1234567890"); // [!code hl]

Output: "0xabcd...7890"
```
