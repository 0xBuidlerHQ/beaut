# address

Formats an `address` into a human-friendly string.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.address("0x1234567890abcdef1234567890abcdef12345678"); // [!code hl]

Output: "0x1234...5678"
```
