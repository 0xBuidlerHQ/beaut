# address

Shortens an EVM address or ENS name into a compact, readable form by trimming the prefix and suffix while leaving ENS names untouched and returning an empty string for invalid input.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const shortAddress = Beaut.address("0x1234567890abcdef1234567890abcdef12345678"); // [!code hl]

Output: "0x1234...5678"
```
