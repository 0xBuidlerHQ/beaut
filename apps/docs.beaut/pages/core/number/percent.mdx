# percent

Creates a reusable percent formatter that enforces two-decimal output via `Intl.NumberFormat` and surfaces a fallback when either the formatter construction or the provided number fails validation.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatPercent = Beaut.percent(); // [!code hl]
const percentString = formatPercent(0.1234);

Output: "12.34%"
```
