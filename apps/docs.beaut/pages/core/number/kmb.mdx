# kmb

Formats numbers into compact K/M/B notation via `Intl.NumberFormat`, while applying safe fallbacks and honoring locale/format overrides.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const compact = Beaut.kmb(1500000); // [!code hl]

Output: "1.50M"
```
