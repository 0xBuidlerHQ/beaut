# bigint

Converts raw `bigint` inputs into human-readable decimal strings through `viem.formatUnits`, while validating decimals and returning a fallback when any argument is malformed.

## Import

```ts twoslash
import { Beaut } from "@0xhq/beaut";
```

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.bigint(1000000000000000000n, 18, { fallback: "0" }); // [!code hl]

Output: "1.0"
```
