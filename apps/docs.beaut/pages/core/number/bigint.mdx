# bigint

Formats raw `bigint` inputs into human-readable decimal strings.

## Usage

```ts twoslash
import { Beaut } from "@0xhq/beaut";

const formatted = Beaut.bigint(1000000000000000000n, 18, { fallback: "0" }); // [!code hl]

Output: "1.0"
```
