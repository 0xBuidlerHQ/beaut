# Installation

Relay is published as `@0xhq/beaut` and ships as ESM.

Make sure your project declares `"type": "module"` or uses a bundler that understands `.ts/.mjs` outputs.

## Add the package

:::code-group
 
```bash [pnpm]
pnpm add @0xhq/beaut
```

```bash [npm]
npm install @0xhq/beaut
```
 
```bash [yarn]
yarn add @0xhq/beaut
```
 
:::
